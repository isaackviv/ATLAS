Getting Started

Bonax -   a Construction & Business HTML Template built with Bootstrap 5 framework. Flat colors, clean and modern design made for Construction website. It is completely easy to use and customize, so friendly, and fully responsive. Check out Live Preview to choose the best Construction Template for making a modern website.

Folders Structure

  • bonax > HTML files.
  • bonax > assets > css> > CSS files.
  • bonax > assets > fonts> > Fonts files.
  • bonax > assets > images> Images files.
  • bonax > assets > js> > JS files.


1. Installation via localhost

Copy and paste template zip file in your local directory.

After that, you should be able to open it in your browser directly.



2. Installation via FTP

Open up your FTP manager and connect to your hosting.

Browse to required directory (Normally public_html).

Upload the files in the folder.
